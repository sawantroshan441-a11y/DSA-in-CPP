# Practical 4 – Tower of Hanoi

## 🎯 Aim

To implement the Tower of Hanoi problem using Recursion in C++.

---

## 📘 Theory

Tower of Hanoi is a famous recursive problem. It consists of three rods and a number of disks of different sizes. The objective is to move all the disks from the source rod to the destination rod using an auxiliary rod.

Rules:
- Only one disk can be moved at a time.
- A larger disk cannot be placed on a smaller disk.
- Only the top disk can be moved.

Recursion is the best approach to solve this problem.

---

## ⚙️ Operations

- Read the number of disks.
- Move disks recursively.
- Display every move.
