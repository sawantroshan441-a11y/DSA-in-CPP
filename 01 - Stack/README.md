# Practical 1 – Stack Operations

## 🎯 Aim
To implement Stack operations (Push, Pop, Peek, and Display) using arrays in C++.

---

## 📚 Theory

A **Stack** is a linear data structure that follows the **LIFO (Last In First Out)** principle. This means the last element inserted into the stack is the first element to be removed.

A stack allows insertion and deletion from only one end, called the **Top**.

---

## ⚙️ Operations

- **Push** – Insert an element into the stack.
- **Pop** – Remove the top element from the stack.
- **Peek** – Display the top element without removing it.
- **Display** – Show all elements present in the stack.

---

## 📝 Algorithm

1. Create an array to store stack elements.
2. Initialize `top = -1`.
3. Display a menu for Stack operations.
4. Read the user's choice.
5. If the choice is **Push**, check for overflow and insert the element.
6. If the choice is **Pop**, check for underflow and remove the top element.
7. If the choice is **Peek**, display the top element.
8. If the choice is **Display**, print all stack elements.
9. Repeat until the user chooses to exit.

---

## ⏱️ Time Complexity

| Operation | Complexity |
|-----------|------------|
| Push | O(1) |
| Pop | O(1) |
| Peek | O(1) |
| Display | O(n) |
